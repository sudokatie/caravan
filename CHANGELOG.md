# Changelog

All notable changes to Caravan.

## [0.1.0] - 2026-02-02

### Added

- Initial release
- Party system with health, status, and daily effects
- Supply management (food, ammo, medicine, spare parts)
- Wagon system with damage, repair, and oxen
- Weather system with seasonal variations
- Random events (illness, injury, theft, breakdown, discovery, animal attack)
- Travel system with pace and terrain modifiers
- River crossing with 4 options (ford, caulk, ferry, wait)
- Hunting minigame
- Store system for buying and selling supplies
- 17 historical locations from Independence to Oregon City
- Full game loop with win/lose conditions
- 262 tests
